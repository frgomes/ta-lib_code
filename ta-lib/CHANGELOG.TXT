V0.1.3 (January 2005)

New Features
============
  - New candlestick functions:
      CDL2CROWS          Two Crows 
      CDL3INSIDE         Three Inside Up/Down 
      CDL3LINESTRIKE     Three-Line Strike 
      CDL3OUTSIDE        Three Outside Up/Down 
      CDL3WHITESOLDIERS  Three Advancing White Soldiers 
      CDLADVANCEBLOCK    Advance Block 
      CDLDARKCLOUDCOVER  Dark Cloud Cover 
      CDLINNECK          In-Neck Pattern 
      CDLONNECK          On-Neck Pattern 
      CDLPIERCING        Piercing Pattern 
      CDLSTALLEDPATTERN  Stalled Pattern 
      CDLTHRUSTING       Thrusting Pattern

  - Excel: Default value for unspecified parameters now handled.
  
  - Excel: Function wizard display the default values.
  
  - .NET Visual Studio 2005 support (Beta)

  - Perl binaries now included in Win32 packages.
